# My Git Project

**En este pequeño proyecto se tratan conceptos como:**

* `git init`
* `git status`
* `git add .`
* `git commit -m "Descripción de la actualualización"`
* Archivos `.gitignore`
* `git remote add origin <URL>` para confingurar el repositorio remoto
* `git pull`
* `git branch`
* `git checkout`
* `git checkout -b nombre-nueva-rama`
* `git rebase`
* `git fetch`
